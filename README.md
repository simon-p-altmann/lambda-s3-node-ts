# AWS Lambda seed project for node typescript

## lambda

cd into lambda dir

### running locally

```
npm i
npm run start
```

### running unit test

```
npm run test
```

with coverage

```
npm run test:unit
```

### lint

```
npm run lint
```

## terraform

cd into terraform dir

## terraform deploy

1. install terraform
2. terraform init
3. terraform apply
4. terraform deploy
